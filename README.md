# NightRatePro

リアルタイムでお会計を管理・予測できる、夜のお店向け会計カウンターアプリです。

## 主な機能

- **リアルタイム会計管理**: 入店からの経過時間に合わせて、セット料金を自動で追加・計算します。
- **お店プリセット保存**: 通い慣れたお店の料金体系（セット料金、ドリンク単価、TAXなど）を複数保存し、ワンタップで切り替え可能です。
- **詳細な注文履歴**: ドリンクや指名などの追加履歴を正確に記録し、いつでも内容を確認・削除できます。
- **お会計予報**: 「あと1セットいたら合計いくらになるか」を最大15セット先まで予測表示します。
- **データバックアップ**: 保存したお店情報や設定をJSONファイルとして書き出し・読み込みできるため、機種変更時のデータ移行も簡単です。
- **結果保存機能**: お会計確定後、詳細な明細をテキストファイルとして保存できます。

## 使い方

1. **お店の設定**: 初回は「店舗名」「セット料金」「TAX」などの基本情報を入力します。
2. **入店**: 「入店」ボタンを押すとカウントが始まります。
3. **項目の追加**: 注文に合わせて「ゲストドリンク」「キャストショット」などのボタン、またはカスタム項目から追加します。
4. **お会計**: 「お会計」ボタンを押すと合計金額が表示され、記録を保存できます。
